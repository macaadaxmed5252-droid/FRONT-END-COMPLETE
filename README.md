# Job Portal Frontend

A modern, responsive job portal application built with React, TypeScript, and Vite. This platform connects job seekers with employers, streamlining the hiring process with a clean and intuitive user interface.

## Features

- **User Roles**: Separate interfaces for Job Seekers and Admins.
- **Job Search**: Advanced filtering and search capabilities.
- **Applications**: Easy application process with tracking.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices.
- **Modern UI**: Built with a custom design system using Tailwind CSS and Radix UI.

## Tech Stack

- **Framework**: React 18
- **Language**: TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **State Management**: Redux Toolkit
- **Routing**: React Router DOM
- **Forms**: React Hook Form
- **Validation**: Zod
- **Icons**: Lucide React

## Getting Started

1. **Install dependencies**:
   ```bash
   npm install
   ```

2. **Start the development server**:
   ```bash
   npm run dev
   ```

3. **Build for production**:
   ```bash
   npm run build
   ```

## Project Structure

- `src/components`: Reusable UI components and design system.
- `src/pages`: Application views and route components.
- `src/store`: Redux state management.
- `src/styles`: Global styles and Tailwind configuration.
